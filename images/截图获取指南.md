# 后台管理端截图获取指南

## 一、启动应用

### 1. 启动后端服务
```bash
cd studyroom-java
mvn spring-boot:run
```
- 后端默认监听 `8081` 端口
- 确保 MySQL 和 Redis 服务已启动

### 2. 启动管理端前端
```bash
cd studyroom-vue
npm install
npm run dev
```
- 访问 http://localhost:9528
- 默认管理员帐号：查看数据库 `sys_admin` 表

## 二、获取截图

### 1. 会员卡管理截图
1. 登录管理后台
2. 在左侧菜单找到 **会员卡管理**（或访问 `/bas/membership-card`）
3. 确保页面显示会员卡列表（月卡、季卡、年卡等）
4. 截取整个页面区域（建议宽度 800-1200px）
5. 保存为：`images/admin-membership-card.png`

### 2. 留言管理截图
1. 在左侧菜单找到 **留言管理**（或访问 `/bas/message`）
2. 确保页面显示用户留言列表
3. 截取整个页面区域
4. 保存为：`images/admin-message.png`

### 3. 楼层管理截图
1. 在左侧菜单找到 **楼层管理**（或访问 `/bas/floor`）
2. 确保页面显示楼层列表（如：1楼、2楼、3楼）
3. 截取整个页面区域
4. 保存为：`images/admin-floor.png`

### 4. 自习室管理截图
1. 在左侧菜单找到 **自习室管理**（或访问 `/bas/studyroom`）
2. 确保页面显示各自习室列表（包含所属楼层、名称、开放时间等）
3. 截取整个页面区域
4. 保存为：`images/admin-studyroom.png`

## 三、截图规范

### 文件命名
- 统一使用 `admin-` 前缀
- 使用小写字母和连字符
- 格式：`admin-{功能名}.png`

### 图片尺寸
- 建议宽度：800-1200px
- 高度：根据内容自适应
- 保持页面完整显示，不要裁切重要信息

### 截图工具建议
- Windows：Snipping Tool、Snip & Sketch
- Mac：Cmd + Shift + 4
- 浏览器插件：Full Page Screen Capture

## 四、测试截图是否正常显示

1. 将截图保存到 `images/` 目录
2. 重新打开 README.md 文件
3. 预览 Markdown 渲染效果
4. 确保图片正常显示且大小合适

## 六、注意事项

1. 确保截图清晰，文字可读
2. 避免截图中包含敏感信息（如密码、手机号等）
3. 保持截图风格一致（相同的浏览器、分辨率）
4. 截图后建议压缩图片大小，保持在 500KB 以内
5. 如果页面数据为空，可以先添加一些测试数据再截图
