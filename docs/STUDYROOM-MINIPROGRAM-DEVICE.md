# 微信小程序：真机调试与体验检查

与 `docs/STUDYROOM-OPERATIONS.md` 中的本地说明配合使用。

## 1. 开发工具与基础设置

- 在微信开发者工具中导入 **`studyroom-wx`** 目录，AppID 使用测试号或正式号。
- **本地设置** 中按需开启「不校验合法域名」（仅开发阶段）；**正式版**必须在公众平台配置 request 合法域名（HTTPS）。

## 2. 本机 / 局域网后端地址

- 模拟器通常可用 `http://127.0.0.1:8081/self-study`（与 `utils/config.js` 中 `baseUrl` 一致）。
- **真机预览** 时手机无法访问电脑的 `127.0.0.1`，请将 `baseUrl` 改为电脑的 **局域网 IP**，例如 `http://192.168.1.10:8081/self-study`，并保证手机与电脑在同一 Wi‑Fi、防火墙放行 **8081**。

## 3. 建议自测路径

1. **首页**：轮播、公告列表或「暂无公告」空态、跳转选座与公告中心。  
2. **预定（region）**：切换楼层后标题区是否显示楼层名；选自习室后是否出现「加载座位」状态；规则区折叠/展开；当日临近打烊时是否出现黄色提示条。  
3. **登录 / 我的**：带 token 时的展示与接口 401 提示。

## 4. 常见问题

| 现象 | 处理方向 |
|------|----------|
| 真机无法请求接口 | 检查 `baseUrl` IP、端口、`/self-study` 路径、防火墙 |
| 合法域名校验失败 | 正式环境配置 HTTPS 域名；开发期可临时关闭校验 |
| 接口 401 | 重新登录；核对管理端与小程序 JWT 配置是否一致 |
